## Icons

| Size | Use |
|------|-----|
| 12px | Inline with small text |
| 16px | Default — most UI contexts |
| 20px | Emphasis, standalone |
| 24px | Large, decorative |

> Icon bounding box is always 16px. Vector inside is typically 13–14px. See spacing.md for optical alignment notes.
